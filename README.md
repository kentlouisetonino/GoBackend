## Go-Labs

> - A laboratory for experimenting with Go concepts.

```bash
# API URL.
http://localhost:11000/api

# Health check route.
GET /health
```

<br />
<br />



## Setup

> - Create a `.env` file.

```sh
# Environment variables.
PORT=11000
```

> - Run the following commands below.

```bash
# Change permission and make it executable.
sudo chmod +x run-build.sh
sudo chmod +x run.sh


# Run the build.
./run-build.sh

# Run the server.
./run.sh
```
